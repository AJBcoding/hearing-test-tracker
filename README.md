# Hearing Test Tracker

A desktop application for visualizing and analyzing audiogram data from hearing tests over time.

## Overview

This tool helps track hearing health by:
- Importing audiogram images via OCR (OpenCV + Tesseract)
- Storing test data in a local database
- Visualizing data in 5 interactive modes
- Analyzing trends and patterns over time
- Generating professional PDF reports

## Technology Stack

- **Backend:** Python (Flask) + PostgreSQL/SQLite
- **Frontend:** React + TypeScript + Mantine UI + Recharts
- **OCR:** OpenCV + Tesseract
- **Deployment:** Local desktop application
